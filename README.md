## 今後の課題(バージョン2.3以降で改善予定)

#### イ「開発者がCSSやJavaScriptに弱い」  
　　開発者は特設サイトにも書いてある通りほとんどCSSとJavaScriptを使ったことがありません。
  
  ### 「たてのさん」はまだまだ進化します。
  
#### ロ「読み込みが遅い」  
　　横書きが中心のWebデザインにおいて、縦書きの専用のフォントを読み込まなければならないので、通常のWebページよりも若干読み込み時間が遅いです。
  
### ！「たてのさん」はMITライセンスに基づくソフトウェアです。  デバックや新機能搭載など、開発の手助けをしてくれる人を随時募集しています。
